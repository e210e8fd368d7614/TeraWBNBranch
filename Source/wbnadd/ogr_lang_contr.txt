-время исполнения кода ограничено максимальным количеством тиков (35000)
-не разрешаются регулярные выражения
-не разрешаются конструкции try-catch (для обеспечения иммутабельности)
-запрещены следующие методы в стандартных объектах:
Array: concat, toString
String: localeCompare, match, repeat, search, padStart, padEnd

Разрешены константы:
true, false, undefined, Infinity, NaN, null

Разрешены следующие функции и переменные:
Стандартные функции: isFinite, isNaN, parseFloat, parseInt, String, Number, Boolean
JSON - стандартный объект серилизации JS
Math - стандартный объект JS для доступа к математическим функциям, исключение: функция random() всегда возвращает 0
this - стандартный объект - контекст исполнения функции
arguments - стандартный объект - массив переменных вызова функции
context - содержит контекст исполнения смарт-контракта, определяется так:
{
   BlockNum,//число - номер текущего блока
   BlockHash,//32-х байтовый массив - хеш текущего блока
   BlockAddrHash,//32-х байтовый массив, в котором id майнера и нонсы для определения хеша блока
   TrNum,//число - номер текущей транзакции
   Account,//объект - текущий счет в контексте которого вызывается смарт-контракт (см. ниже описание)
   Smart,//объект - текущий смарт-контракт (см. ниже описание)
   FromNum,//число - номер счета отправителя
   ToNum,//число - номер счета получателя
   Description,//строка - описание платежа
   Value:{SumCOIN /*число целых монет*/,SumCENT/*число дробных монет*/},//сумма платежа
};

Описание полей объектов:

Объект счет Account задается так:
{
 	Num, //число - номер счета
Currency, //число - номер валюты
PubKey, //33-х байтовый массив - публичный ключ
Name,//строка - наименование счета
BlockNumCreate, // число - номер блока когда был создан счет
Adviser, // число - номер счета адвайзера
Value:{SumCOIN /*число целых монет*/,SumCENT/*число дробных монет*/, OperationID /*число - номер операции*/,Smart /*число - номер смарт-контракта*/},
}

Объект смарт-контракт Smart задается так:
{
 	Num, //число - номер счета
Version, //число - версия смарт-контакта
TokenGenerate, //число - признак что смарт-контракт создает новую валюту в системе
BlockNum, //число - номер блока в котором создан смарт-контракт
TrNum, //число - номер транзакции в которой создан смарт-контракт
Owner, //число - номер-счета отправителя транзакции, которая создала счет (по умолчанию это не дает никаких дополнительных прав)
Account, //число - номер базового счета смарт-контракта
AccountLength, //число - количество счетов начиная от базового (1-50)
ISIN, //строка 12 символов (International Securities Identification Numbers — международный идентификационный код ценной бумаги)
ShortName, //строка 5 символов - краткое название токена (для отображения)
Name, //строка - имя смарт-контракта
Description, //строка - описание
Code, //строка - код на javascript
HTML, //строка - код интерфейса в HTML
StateFormat, //строка - формат полей состояний
}
